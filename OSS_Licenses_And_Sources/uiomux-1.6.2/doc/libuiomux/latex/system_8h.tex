\subsection{Detailed Description}
UIOMux system API. These functions are used by the commandline tool uiomux, and should not be called by normal applications.

\subsubsection[{uiomux\_\-system\_\-destroy}]{\setlength{\rightskip}{0pt plus 5cm}int uiomux\_\-system\_\-destroy (UIOMux $\ast$ {\em uiomux})}\label{system_8h_a5973bb40a90b5f3a813157ac0ac5a341}


Destroy the UIOMux system, removing the associated shared memory segment and destroying all shared mutexes. 

This will make UIOMux unusable by other applications which have previously opened it, so must not be used by normal applications. It is usually called by the commandline tool 'uiomux destroy'.

Reset the UIOMux system, initializing the associated shared memory segment and all shared mutexes. 

Note that this is done transparently by the first process to call \doxyref{uiomux\_\-open()}{p.}{uiomux_8h_a71e45d7bbd218556dd55d79e153fee1c}, so this function does not need to be used by normal applications. It is usually called by the commandline tool 'uiomux reset'.
